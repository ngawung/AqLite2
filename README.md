# Cetera Launcher
![logo](https://i.ibb.co/ct7psQ2/20210207-055559.png)

Welcome to [AquaStar](https://github.com/aquaspy/AquaStar) fork <strong>Cetera Launcher</strong>

this is a Cetera launcher used for botting in Adventure Quest World Game. Use with your own risk!

Feeling lost? looking for AqLite Launcher instead? no worries this launcher also include AqLite

This launcher contain 2 game client, <strong>Cetera Bot</strong> and <strong>AqLite</strong>

Aqlite swf are from this [AQLite repository](https://github.com/133spider/AQLite)<br>
while Cetera Bot swf are dump from flash projector using this [dump_projector](https://github.com/laenion/dump_projector)

## How to download

no download link to prevent ordinary people using it. build it manually :)

## How to build
 - clone this repository
 - install npm dependency
 - run `npm run dist`

to dump cetera swf run `npm run dump`
